# video 目录说明

本目录用于保存 iDrug 平台 demo 视频相关材料，包括视频链接、演示脚本、录制说明和剪辑记录。

## 建议内容

- `video_link.md`：demo 视频链接、访问方式、提取码或课程平台提交说明。
- `demo_script.md`：演示脚本，按操作顺序描述每一步展示内容。
- `recording_notes.md`：录制环境、浏览器、分辨率、注意事项和重录记录。

## Demo 视频建议流程

1. 打开在线部署地址或本地运行地址。
2. 展示首页和主要功能入口。
3. 演示 DTI 预测输入和结果展示。
4. 演示多模型预测任务。
5. 演示个性化药物推荐与重排序结果。
6. 展示数据浏览、结构查询或用户中心等辅助功能。
7. 简要说明项目架构、模型权重和数据库初始化方式。

## 提交注意事项

- 不建议直接提交 `.mp4`、`.avi`、`.mov` 等大视频文件到 GitHub。
- 推荐在本目录提交视频链接和脚本，视频文件通过课程平台、网盘或压缩包提供。
- 视频画面中不要暴露真实服务器密码、数据库密码、API Key、Token、个人隐私信息。
- 如果链接包含提取码或访问限制，请确认课程评审人员可以正常访问。
